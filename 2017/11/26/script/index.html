比较符号

    条件?语句1:语句2

js作用域及变量

  • 作用域：只有函数带有作用域，一个函数就是一个作用域
  • 全局变量：在整个js中都可以被使用的变量，在全局作用域中声明的变量
  • 局部变量：只可以在局部作用域中使用的变量；在函数（局部作用域）声明的变量就是局部变量
